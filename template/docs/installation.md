# Installation

## Stable release

To install Python Boilerplate, run this command in your terminal:

```sh
uv add template
```

Or if you prefer to use `pip`:

```sh
pip install template
```

## From source

The source files for Python Boilerplate can be downloaded from the [Github repo](https://github.com/xun_6666@163.com/python_boilerplate).

You can either clone the public repository:

```sh
git clone https://github.com/xun_6666@163.com/python_boilerplate
```

Or download the [tarball](https://github.com/xun_6666@163.com/python_boilerplate/tarball/main):

```sh
curl -OJL https://github.com/xun_6666@163.com/python_boilerplate/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd template
uv sync
```
