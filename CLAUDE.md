# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

### Building
- To build the codebase, use the following command: `bash build.sh`

### Linting
- For linting the code, use: `bash lint.sh`

### Running Tests
- Run tests with: `bash test.sh`

### Running a Single Test
- To run a specific test, use: `bash test.sh --test <test-name>`

## Code Architecture and Structure

- The codebase is structured as follows:
  - `src/`: Contains source files
  - `tests/`: Contains test files
  - `docs/`: Documentation files
  - `template/`: Template files

- For high-level architecture details, refer to the README.md file.
